## How to setup for Tests
1. Clone repo
```bash
git clone https://github.com/mibal-ua/testing-lab2.git
```
2. Go to project folder
3. Install mvn dependencies
```bash
mvn clean install 
```
4. Run tests
```bash
mvn test 
```
